# Stay Compass OS

Version: 0.1.0

## Purpose

Stay Compass OS is a secure operating system for Raspberry Pi devices that boot directly into the Stay Compass experience.

The operating system is designed to be remotely managed, easy to deploy, and locked down so guests cannot access the underlying operating system.

## Goals

- Boot directly into Stay Compass
- Simple deployment
- Automatic updates
- Secure kiosk mode
- Easy Wi-Fi provisioning
- Factory reset capability
- Support future remote management through Gulera or another approved management service
